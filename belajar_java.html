Dasar-Dasar Java

15 Elemen dan Fungsi Dasar Bahasa Pemrograman Java

1. Hello World

Program pertama untuk mencetak teks ke layar.

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, altap"); } }

2. Variabel

Digunakan untuk menyimpan data.

int umur = 15; String nama = "Andi";

3. Tipe Data

Beberapa tipe data utama dalam Java.

int angka = 10; float nilai = 7.5f; boolean aktif = true; char huruf = 'A';

4. Percabangan

Pengambilan keputusan menggunakan if-else.

if (umur >= 18) { System.out.println("Dewasa"); } else { System.out.println("Anak-anak"); }

5. Switch

Mengganti if-else bertingkat.

int hari = 3; switch (hari) { case 1: System.out.println("Senin"); break; case 2: System.out.println("Selasa"); break; default: System.out.println("Hari lainnya"); }

6. Perulangan For

Digunakan untuk mengulang kode dalam jumlah tertentu.

for (int i = 0; i < 5; i++) { System.out.println("Perulangan ke-" + i); }

7. While Loop

Perulangan selama kondisi terpenuhi.

int i = 0; while (i < 5) { System.out.println(i); i++; }

8. Do-While Loop

Minimal dieksekusi sekali, lalu cek kondisi.

int i = 0; do { System.out.println(i); i++; } while (i < 5);

9. Array

Struktur data untuk menyimpan banyak nilai.

String[] buah = {"apel", "jeruk", "mangga"}; System.out.println(buah[1]);

10. Method

Blok kode yang bisa dipanggil berkali-kali.

public static void sapa(String nama) { System.out.println("Halo, " + nama); } sapa("Rani");

11. Class dan Object

Dasar dari OOP (Pemrograman Berorientasi Objek).

class Siswa { String nama = "Budi"; } Siswa s = new Siswa(); System.out.println(s.nama);

12. Konstruktor

Fungsi khusus saat objek dibuat.

class Mobil { Mobil() { System.out.println("Mobil dibuat!"); } } Mobil m = new Mobil();

13. Inheritance

Konsep pewarisan sifat dari class.

class Hewan { void suara() { System.out.println("Suara hewan"); } } class Kucing extends Hewan {} Kucing k = new Kucing(); k.suara();

14. Interface

Template kontrak class-class lain.

interface Hewan { void suara(); } class Anjing implements Hewan { public void suara() { System.out.println("Guk Guk"); } }

15. Try-Catch

Menangani error atau exception.

try { int hasil = 10 / 0; } catch (Exception e) { System.out.println("Terjadi kesalahan: " + e); }

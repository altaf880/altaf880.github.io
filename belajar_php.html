Dasar-Dasar PHP

Pelajari elemen penting dalam pemrograman PHP

1. Tag Dasar PHP

Tag pembuka dan penutup PHP digunakan untuk menulis kode PHP di dalam file HTML.

<?php echo "Halo Dunia!"; ?>

2. Variabel

Digunakan untuk menyimpan nilai.

<?php $nama = "altap"; $umur = 15; echo $nama; ?>

3. Tipe Data

PHP memiliki tipe data seperti string, integer, float, boolean, array, dan object.

<?php $teks = "Hello"; $angka = 10; $desimal = 3.14; $benar = true; ?>

4. Percabangan (if/else)

Digunakan untuk pengambilan keputusan.

<?php $nilai = 75; if ($nilai >= 70) { echo "Lulus"; } else { echo "Tidak Lulus"; } ?>

5. Perulangan

Untuk mengulang eksekusi kode.

<?php for ($i = 0; $i < 5; $i++) { echo "Angka $i
"; } ?>

6. Array

Menyimpan banyak nilai dalam satu variabel.

<?php $buah = array("apel", "jeruk", "mangga"); echo $buah[1]; ?>

7. Function

Membuat blok kode yang bisa dipanggil berulang.

<?php function sapa($nama) { echo "Halo, $nama"; } sapa("altap"); ?>

8. Superglobal ($_GET, $_POST)

Digunakan untuk mengambil data dari form atau URL.

<form method="post"> Nama: <?php if ($_SERVER["REQUEST_METHOD"] == "POST") { $nama = $_POST["nama"]; echo "Halo, $nama"; } ?>

9. Include dan Require

Menyisipkan file PHP lain ke dalam file utama.

<?php include 'header.php'; require 'config.php'; ?>

10. Class dan Object

Untuk menerapkan konsep OOP.

<?php class Mobil { public $warna; function set_warna($warna) { $this->warna = $warna; } } $avanza = new Mobil(); $avanza->set_warna("Merah"); echo $avanza->warna; ?>

11. Try-Catch

Menangani error dengan pengecualian.

<?php try { throw new Exception("Terjadi kesalahan!"); } catch (Exception $e) { echo $e->getMessage(); } ?>

12. Koneksi ke Database

Menghubungkan PHP ke MySQL.

<?php $conn = mysqli_connect("localhost", "root", "", "dbku"); if (!$conn) { die("Koneksi gagal: " . mysqli_connect_error()); } ?>

13. CRUD - Create

Menyimpan data ke database.

<?php $sql = "INSERT INTO siswa (nama, umur) VALUES ('altap', 15)"; mysqli_query($conn, $sql); ?>

14. CRUD - Read

Menampilkan data dari database.

<?php $result = mysqli_query($conn, "SELECT * FROM siswa"); while($row = mysqli_fetch_assoc($result)) { echo $row['nama'] . "
"; } ?>

15. CRUD - Update & Delete

Update dan hapus data dari database.

<?php mysqli_query($conn, "UPDATE siswa SET umur=19 WHERE nama='Budi'"); mysqli_query($conn, "DELETE FROM siswa WHERE nama='altap'"); ?>

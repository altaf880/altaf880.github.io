Panduan Dasar JavaScript

Berikut adalah elemen-elemen dasar JavaScript dan fungsinya, lengkap dengan contoh penggunaannya:

let / const / var

Digunakan untuk mendeklarasikan variabel.

let nama = "Altaf"; const umur = 20; var kota = "Jakarta";

function

Fungsi adalah blok kode yang bisa digunakan ulang.

function sapa(nama) { console.log("Halo, " + nama); } sapa("Altaf");

if / else

Untuk membuat logika percabangan.

if (umur > 18) { console.log("Dewasa"); } else { console.log("Remaja"); }

for / while

Melakukan perulangan kode.

for (let i = 0; i < 3; i++) { console.log("Ulang ke-" + i); }

document.getElementById()

Memilih elemen berdasarkan ID.

let judul = document.getElementById("judul"); judul.textContent = "Judul Baru";

addEventListener

Menangani event pengguna seperti klik.

dokumen.getElementById("tombol") .addEventListener("click", function() { alert("Tombol diklik!"); });

console.log()

Menampilkan output ke console (debugging).

console.log("Ini debug");

Array & Object

Struktur data penting dalam JavaScript.

let buah = ["apel", "jeruk"]; let orang = { nama: "Altaf", umur: 20 };

setTimeout / setInterval

Menunda atau mengulang fungsi dalam waktu tertentu.

setTimeout(() => console.log("Tunggu 2 detik"), 2000); setInterval(() => console.log("Setiap 1 detik"), 1000);

fetch()

Mengambil data dari server (API).

fetch("https://api.example.com/data") .then(res => res.json()) .then(data => console.log(data));

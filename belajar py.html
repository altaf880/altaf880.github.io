Dasar-Dasar Python

Pelajari elemen dasar Python beserta contoh penggunaannya

1. Variabel

Digunakan untuk menyimpan data.

x = 5 y = "Halo" print(x, y)

2. Tipe Data

Contoh tipe data: Integer, String, Float, Boolean.

umur = 15 nama = "Altag" tinggi = 1.60 aktif = True

3. Percabangan (If)

Menjalankan kode berdasarkan kondisi tertentu.

nilai = 80 if nilai > 75: print("Lulus") else: print("Tidak Lulus")

4. Perulangan (For dan While)

Melakukan perulangan beberapa kali.

for i in range(3): print("Halo") n = 0 while n < 3: print("Hi") n += 1

5. Fungsi

Blok kode yang dapat dipanggil berulang.

def sapa(nama): print("Halo,", nama) sapa("altap")

6. List (Array)

Struktur data untuk menyimpan banyak nilai.

buah = ["apel", "pisang", "jeruk"] print(buah[0])

7. Dictionary

Penyimpanan data dengan pasangan key-value.

mahasiswa = {"nama": "altap", "umur": 15} print(mahasiswa["nama"])

8. Tuple

Mirip list, tapi tidak dapat diubah (immutable).

warna = ("merah", "hijau", "biru") print(warna[1])

9. Set

Kumpulan item unik, tidak memiliki indeks.

angka = {1, 2, 3, 3, 2} print(angka)

10. Class dan Object

Membuat objek dengan atribut dan metode.

class Orang: def __init__(self, nama): self.nama = nama def sapa(self): print("Halo, saya", self.nama) siswa = Orang("altap) siswa.sapa()

11. Try-Except

Menangani error agar program tidak berhenti tiba-tiba.

try: hasil = 10 / 0 except ZeroDivisionError: print("Tidak bisa dibagi dengan nol")

12. Import Modul

Memasukkan fungsi dari modul lain.

import math print(math.sqrt(16))
